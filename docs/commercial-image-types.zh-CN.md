# 商业化图片类型调研

这份笔记总结了产品营销、品牌 campaign、电商与内容分发里常见的商业化图片类型。

## 为什么现在这些图会越来越像

很多 prompt 库会过度集中在这几类：

- 白底或纯净背景的单品主图
- 近距离产品英雄图
- 奢侈品静物陈列

这些当然有用，但它们只覆盖了商业需求的一小部分。真实品牌通常需要的是一整套图像系统，而不是一种图拍到底。

## 常见商业图片类型

### 1. 单品目录图

常见场景：

- PDP 主图
- 电商平台主图
- 零售目录图

作用：

- 讲清楚产品
- 建立信任
- 方便比较

### 2. 细节 / 微距图

常见场景：

- 工艺证明
- 材质与做工表达
- 高端产品转化支持

作用：

- 展示缝线、纹理、切面、五金、材质、功能细节

### 3. 尺寸 / 对比说明图

常见场景：

- 解释大小
- 帮助用户理解比例
- 降低购买不确定性

作用：

- 通过熟悉参照物、人物、空间或包装建立尺度感

### 4. 生活方式 / 使用场景图

常见场景：

- 帮用户想象拥有感
- 呈现使用方式
- 把功能翻译成情绪

作用：

- 让产品出现在可信的真实场景中

### 5. 系列 / 组合图

常见场景：

- 展示整套产品线
- 比较颜色、型号、口味、尺寸
- 呈现系列关系

作用：

- lineup、assortment、bundle、gift set

### 6. Flat Lay / Laydown 故事图

常见场景：

- 编辑感内容
- 成分故事
- “包里有什么” 或生活方式内容

作用：

- 俯拍构图，便于强控制和讲故事

### 7. 包装 / 开箱图

常见场景：

- 上新发布
- 节庆送礼
- 创作者社媒内容

作用：

- 强化包装设计
- 建立仪式感和期待感

### 8. 户外广告 / OOH 主视觉

常见场景：

- 品牌发布
- 橱窗画面
- 地铁、机场、街头广告位 mockup

作用：

- 让品牌从“商品图”升级成“传播图”

### 9. 社媒 Story / 竖版广告图

常见场景：

- 付费投放
- 新品快闪传播
- 移动端社媒版位

作用：

- 强化竖屏阅读效率
- 为文案与按钮预留空间

### 10. 编辑拼贴 / Campaign Board

常见场景：

- 品牌定调
- 系列发布
- 创意提案或 moodboard

作用：

- 把产品、细节、材质、场景和图形节奏组合成系统

### 11. Brochure / Catalog / Review Graphic

常见场景：

- 产品册
- 销售页
- 测评图卡
- 内部提案

作用：

- 图片需要和信息模块共存，而不是只负责“好看”

## 这对仓库意味着什么

一个成熟的 GPT-Image-2 商业 prompt 库，至少应该覆盖 3 层：

1. 转化素材
2. 品牌建设素材
3. 分发渠道素材

当前仓库已经覆盖了第 1 层和部分第 2 层，接下来最该补的是第 3 层：

- 竖版 Story 广告
- 包装开箱图
- OOH 主视觉
- 编辑拼贴 campaign
- 尺度说明图
- 适合 brochure 的信息型画面
