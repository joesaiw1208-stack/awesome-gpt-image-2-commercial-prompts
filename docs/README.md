## Why These Files Exist

- `gpt-image-2-best-practices*.md`
  Helps contributors write prompts in a consistent structure aligned with current OpenAI image guidance.
- `commercial-image-types*.md`
  Explains the business logic behind different commercial asset types, so the repo stays commercially useful rather than becoming a random image gallery.
- `by-industry.md`
  Helps users browse by vertical such as beauty, fashion, jewelry, hospitality, automotive, and consumer tech.
- `by-use-case.md`
  Helps users browse by business intent such as product display, brand visuals, social ads, explainers, or trust-building assets.
- `by-format.md`
  Helps users browse by asset format such as billboard, story frame, unboxing, lookbook, and infographic.
