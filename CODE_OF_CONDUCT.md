# Code of Conduct

## Our Pledge

We want this project to be welcoming, constructive, and useful to contributors from design, engineering, marketing, and creative backgrounds.

## Expected Behavior

- be respectful
- give actionable feedback
- credit ideas and sources when possible
- focus on improving prompts, documentation, and examples

## Unacceptable Behavior

- harassment or discrimination
- personal attacks
- spammy self-promotion
- submitting copied work without context

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that harm the health of the project.
