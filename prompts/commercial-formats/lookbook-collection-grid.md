# Lookbook Collection Grid

## Best For

- collection launch page
- brand lookbook
- category landing page visual

## Prompt (EN)

```text
Goal: Create a commercial lookbook-style composition that presents a collection rather than a single SKU.
Asset: Collection landing page hero or seasonal brand lookbook cover.
Scene: Clean editorial system combining multiple products or looks in a curated grid-like arrangement with premium spacing and strong art direction.
Subject: A coordinated collection shown through several related frames, crops, or silhouettes that feel unified.
Composition: Grid or modular composition with consistent rhythm, variation in crop scale, and clear collection storytelling.
Lighting: Cohesive image treatment across all panels so the set feels intentionally art-directed.
Materials and styling: Editorial lookbook aesthetic, premium paper or campaign feel, disciplined palette, elevated retail fashion or lifestyle tone.
Constraints: No messy collage chaos, no unrelated props, no watermark.
```

## 提示词（中文）

```text
目标：生成一张强调“系列感”而不是“单品感”的商业 lookbook 构图。
资产类型：系列落地页 hero、品牌 lookbook 封面或季节性发布图。
场景：通过多个产品或多个造型画面组成一个克制的编辑系统，具有高级间距和 art direction。
主体：一组彼此协调的产品、造型或画面裁切，共同传达同一套 collection 气质。
构图：网格或模块化布局，节奏统一，裁切比例有变化，能清晰讲出系列故事。
光线：所有模块都要保持统一的图像处理和高级调性。
材质与风格：编辑式 lookbook 审美、像纸质 campaign 或零售品牌系统，配色克制。
约束：不要混乱拼贴，不要无关道具，不要水印。
```
