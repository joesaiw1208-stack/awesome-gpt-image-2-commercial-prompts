# Scale Comparison Explainer

## Best For

- size explanation visual
- product education asset
- brochure or PDP secondary image

## Prompt (EN)

```text
Goal: Create a polished commercial explainer image that communicates product scale clearly without looking like a bland technical diagram.
Asset: Size or proportion explainer for a product page, brochure, or sales deck.
Scene: Clean minimal environment with one familiar reference object or believable context anchor that helps communicate size.
Subject: One product shown clearly alongside a carefully chosen scale reference, with the product still remaining visually premium.
Composition: Straightforward, readable composition with space for labels if needed, product dominant, reference object supportive, not competing.
Lighting: Clear commercial lighting with enough depth to keep the scene elegant and legible.
Materials and styling: Modern explainer aesthetic, clean surfaces, disciplined palette, premium but informative mood.
Constraints: No clutter, no childish infographic style, no watermark, no confusing perspective distortion.
```

## 提示词（中文）

```text
目标：生成一张既能清楚解释产品尺寸，又不显得枯燥技术化的商业说明图。
资产类型：产品页、销售册或 brochure 用的尺寸 / 比例说明视觉。
场景：干净极简的环境里加入一个熟悉参照物或可信场景锚点，用来帮助理解尺度。
主体：一个主产品和一个精心选择的尺寸参照共同出现，但产品依旧要保持高级感。
构图：信息清楚、易读，可为标注预留空间，产品是主角，参照物只是辅助。
光线：清晰商业布光，既保证可读性，也保留一定高级层次。
材质与风格：现代说明图审美、干净表面、克制配色，信息化但不廉价。
约束：不要杂乱，不要幼稚信息图风格，不要水印，不要令人困惑的透视变形。
```
