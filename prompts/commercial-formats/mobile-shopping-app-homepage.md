# Mobile Shopping App Homepage

## Best For

- ecommerce UI concept
- marketplace campaign mockup
- app launch screenshot

## Prompt (EN)

```text
Goal: Create a high-fidelity mobile shopping app homepage screenshot that feels commercially plausible and conversion-ready.
Asset: Mobile ecommerce interface mockup for a campaign launch or product design showcase.
Scene: Realistic smartphone app homepage with search, hero promo banner, category shortcuts, flash-sale module, and double-column product feed.
Subject: One polished ecommerce home screen designed around browsing, promotion, and product discovery rather than abstract UI decoration.
Composition: Full mobile portrait screenshot, edge-to-edge interface, clean information hierarchy, realistic spacing, strong retail logic, no cropped modules.
Lighting: Flat screen capture realism with subtle glossy-device polish only if helpful.
Materials and styling: Modern ecommerce UI, card shadows, rounded modules, commercial Chinese-marketplace energy, promo red accents balanced with clean white space.
Constraints: No broken icons, no unreadable text blocks, no fantasy UI, no excessive clutter, no watermark.
```

## 提示词（中文）

```text
目标：生成一张高保真的移动电商首页截图，既要真实可信，又要具备商业转化思路。
资产类型：电商 App 首页 mockup，用于活动预热、产品设计展示或营销素材。
场景：真实的手机电商首页，包含搜索区、主促销 banner、分类快捷入口、限时秒杀模块和双列商品流。
主体：一张围绕“浏览、促销、发现商品”组织的完整首页界面，而不是抽象 UI 装饰图。
构图：完整手机竖版截图，界面铺满画布，信息层级清晰，间距真实，零售逻辑完整，不要裁掉关键模块。
光线：以真实屏幕截图感为主，若需要可带极轻微设备高光质感。
材质与风格：现代电商 UI、卡片阴影、圆角模块、带中国 marketplace 气质的商业设计，促销红点缀与留白平衡。
约束：不要图标错误，不要大段不可读文字，不要幻想式 UI，不要过度拥挤，不要水印。
```
