# Editorial Fashion Flatlay

## Use Case

Category page visual, social commerce asset, style collection launch, and fashion email banner.

## Asset Type

Flatlay campaign visual

## Prompt

```text
Use case: product-mockup
Asset type: fashion flatlay campaign image
Primary request: Create a premium editorial flatlay for a modern fashion drop with a hero garment and carefully selected accessories.
Scene/backdrop: refined textured fabric backdrop in soft stone gray, minimal shadows, tidy styling area.
Subject: one hero garment neatly arranged with complementary accessories, contemporary fashion brand aesthetic, premium but wearable.
Style/medium: fashion e-commerce editorial photography.
Composition/framing: top-down flatlay, magazine-level styling discipline, balanced negative space, product-first layout.
Lighting/mood: soft diffused overhead lighting, elegant, fresh, brand-safe.
Color palette: neutral monochrome with one accent color.
Materials/textures: cotton, linen, leather, brushed metal accessory details.
Text (verbatim): ""
Constraints: no messy styling, no extra props unrelated to fashion, no text, no watermark, no warped garments.
Avoid: fast-fashion clutter, chaotic influencer aesthetic, low-resolution fabric detail.
```

## Why It Works

It gives teams a repeatable fashion prompt that feels editorial without becoming inaccessible or visually noisy.
